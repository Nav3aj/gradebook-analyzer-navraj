# Gradebook Analyzer

**Author:** Navraj Amgai  
**Date:** 2025-11-20  
**Course:** ETCCPP102 - Programming for Problem Solving Using Python

## Description
CLI tool to load student marks manually or from CSV, compute average/median/min/max, assign letter grades (A–F), list pass/fail students, and export results.

## Files
- `gradebook.py` — main program
- `students.csv` — example input CSV (optional)
- `sample_runs.txt` — sample runs demonstrating program output
